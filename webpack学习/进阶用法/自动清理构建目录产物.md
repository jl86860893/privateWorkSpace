每次构建前清理output目录
## 通过npm script 清理构建目录(手动)
rm -rf ./dist && webpack
rimraf ./dist && webpack

## 自动清理
使用clean-webpack-plugin
默认会删除output指定的输出目录
>$ npm i clean-webpack-plugin -D
```js
const CleanWebpackPlugin = require('clean-webpack-plugin')

module.exports = {
  entry: {
    app: './src/app.js',
    search: './src/search.js',
  },
  output: {
    filename: '[name][chunkhash:8].js',
    path: __dirname + '/dist'
  },
  plugins: [
    new CleanWebpackPlugin()
  ]
}
```
